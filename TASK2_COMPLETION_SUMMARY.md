```

返回分析结果 → 前端渲染卡片流
```

## 🚀 使用方式

### 1. 添加自选股
1. 导航到Portfolio页面
2. 在"自选股管理"标签页中搜索股票
3. 点击"添加"按钮将股票加入自选股

### 2. 运行AI深度推演
1. 确保有自选股
2. 点击"⚡️ AI深度推演"按钮
3. 等待AI分析完成（约10-30秒）
4. 查看分析结果卡片

### 3. 查看分析结果
- **高风险警告**: 红色卡片，建议谨慎操作
- **中等风险**: 黄色卡片，建议关注风险
- **低风险**: 默认卡片，相对安全
- **交易建议**: 具体的买入/卖出/持有建议
- **逻辑链**: AI的分析推理过程
