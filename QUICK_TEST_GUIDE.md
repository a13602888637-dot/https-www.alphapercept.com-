## 🔄 清除缓存（必须！）

### Windows/Linux:
1. 按 `Ctrl + Shift + Delete`
2. 或者按 `Ctrl + Shift + R` (强制刷新)

### Mac:
1. 按 `Cmd + Shift + Delete`
2. 或者按 `Cmd + Shift + R` (强制刷新)

### 或者:
1. 打开开发者工具 (F12)
2. 右键点击刷新按钮
3. 选择"清空缓存并硬性重新加载"

## ✅ 测试步骤

### 步骤1: 登录

1. 访问 https://www.alphapercept.com
2. 点击登录
3. 使用您的账号登录
4. **确认右上角显示您的用户名/头像**

### 步骤2: 进入自选股页面

1. 点击左侧菜单"自选股"
2. 或访问 https://www.alphapercept.com/watchlist

### 步骤3: 添加股票

1. 点击右上角"添加股票"按钮
2. 会弹出对话框

**方法A - 使用搜索（推荐）**:
```
1. 在搜索框输入: 000001
2. 等待1秒，会出现下拉列表
3. 点击列表中的"平安银行"
4. 代码和名称会自动填入下方输入框
5. （可选）填写买入成本等
6. 点击"添加"按钮
7. 应该显示绿色成功提示
8. 股票出现在列表中
```

**方法B - 手动输入**:
```
1. 忽略搜索框
2. 直接在"股票代码"框输入: 600519
3. 直接在"股票名称"框输入: 贵州茅台
4. （可选）填写买入成本等
5. 点击"添加"按钮
6. 应该显示绿色成功提示
7. 股票出现在列表中
```

## 📋 预期行为

### 成功的情况:

1. 点击"添加"后：
   - 显示"正在添加到自选股..."（加载提示）
   - 1-2秒后显示"✅ [股票名] ([代码]) 已添加到自选股"
   - 对话框自动关闭
   - 列表中出现新添加的股票

2. 失败的情况（正常的失败）:
   - 如果未登录：显示"❌ 请先登录以添加自选股"
   - 如果重复添加：显示"⚠️ 该股票已在自选股中"
   - 如果未填写：显示"请输入股票代码和名称"

### 不正常的情况:

- 点击没有任何反应
- 一直显示加载中
- 显示其他错误信息
