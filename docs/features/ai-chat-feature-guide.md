# AI 对话功能 - 使用指南

## 功能概述

DeepSeek AI 实时对话功能为每个股票提供专业的投资分析和建议，支持流式响应、自定义提问和对话历史管理。

---

## 功能入口

### 访问路径
1. **从自选股列表进入**
   - 首页 → 自选股 → 点击任意股票
   - 滚动到 "AI智能分析" 卡片
   - 点击 "💬 AI对话" 标签页

2. **直接访问股票详情**
   - URL: `/stocks/{stockCode}`
   - 例如: `/stocks/600519` (贵州茅台)

## 使用方式

### 方式一: 预设模板（推荐）

**操作步骤**:
1. 点击预设模板按钮（如 "📊 技术分析"）
2. AI 自动开始分析
3. 观察流式响应逐字展示

**预设模板列表**:

| 模板 | 用途 | 示例问题 |
|-----|------|---------|
| 📊 技术分析 | K线和指标分析 | "请基于当前K线走势和技术指标，分析贵州茅台的技术面情况和短期走势预测。" |
| 🎯 买卖建议 | 操作建议 | "结合MA60均线、MACD和RSI指标，给出贵州茅台当前的买卖建议和止损位。" |
| ⚠️ 风险评估 | 风险识别 | "分析贵州茅台当前存在的主要风险点，包括技术风险、市场风险和基本面风险。" |
| 💡 投资策略 | 策略规划 | "基于五大投资流派，为贵州茅台制定短期和中期的投资策略。" |

### 方式二: 自定义提问

**操作步骤**:
1. 在输入框输入问题
2. 按 Enter 键发送（Shift+Enter 换行）
3. 等待 AI 回复

**提问技巧**:
- ✅ 具体明确: "当前价格是否适合买入？"
- ✅ 有针对性: "分析近期成交量变化的意义"
- ✅ 多维度: "对比同行业其他股票的优劣势"
- ❌ 过于宽泛: "这个股票怎么样？"
- ❌ 非股票问题: "今天天气如何？"

## 功能特性

### 1. 流式响应

**效果**: AI 回复逐字展示，类似打字机效果

**性能指标**:
- 首字延迟: ~85ms
- 流式速度: ~11.6 chunks/s
- 平均响应时间: 2-5秒

**用户体验**:
- 实时反馈，减少等待焦虑
- 可立即看到部分答案
- 流畅的动画效果

### 2. 对话历史

**存储机制**:
- 本地存储 (localStorage)
- 每个股票独立保存
- 最多保存 20 条消息
- 刷新页面不丢失

**管理操作**:
- 自动保存: 无需手动操作
- 清除历史: 点击 "清除对话" 按钮
- 查看历史: 打开对话即可看到

### 3. 上下文感知

**自动注入信息**:
```
- 股票代码: 600519
- 股票名称: 贵州茅台
- 当前价格: ¥1680.00
- 涨跌幅: +2.5%
```

**优势**:
- AI 回答更准确
- 无需重复提供股票信息
- 自动关联实时数据

### 4. 错误处理

**错误类型及处理**:

| 错误类型 | 显示方式 | 用户操作 |
|---------|---------|---------|
| API 密钥错误 | 错误提示 | 联系管理员 |
| 网络超时 | 错误消息 | 重新发送 |
| API 限流 | 提示等待 | 稍后重试 |
| 解析错误 | 忽略继续 | 无需操作 |

**错误提示示例**:
```
抱歉，发生了错误：API请求失败

请检查：
1. DeepSeek API密钥是否配置正确
2. 网络连接是否正常
3. API服务是否可用
```

## 最佳实践

### ✅ 推荐做法

1. **首次使用预设模板**
   - 了解 AI 分析风格
   - 获得结构化答案
   - 建立基础认知

2. **结合实时数据提问**
   - 关注当前价格变化
   - 询问特定技术指标
   - 分析突发事件影响

3. **定期清理历史**
   - 避免混淆不同时期的分析
   - 释放存储空间
   - 保持对话整洁

4. **多维度验证**
   - AI 建议 + 自己判断
   - 结合其他信息源
   - 不盲目依赖单一建议

### ❌ 避免做法

1. **不要完全依赖 AI**
   - AI 不能预测未来
   - 仅作为参考工具
   - 投资决策需自主判断

2. **不要频繁切换股票**
   - 对话历史独立存储
   - 切换股票会丢失上下文
   - 完成一个股票分析后再切换

3. **不要发送敏感信息**
   - 对话记录本地存储
   - 不要输入个人信息
   - 不要透露账户数据

## 性能优化建议

### 加快响应速度

1. **使用预设模板**
   - 预设问题经过优化
   - 响应速度更快
   - 答案更结构化

2. **简化提问**
   - 避免过长问题
   - 一次问一个重点
   - 拆分复杂问题

3. **网络优化**
   - 使用稳定网络
   - 避免高峰期使用
   - 关闭不必要的网络应用

### 减少错误

1. **检查环境**
   - 确认网络连接
   - 验证 API 密钥
   - 更新浏览器版本

2. **合理使用**
   - 避免过度频繁请求
   - 等待上一条回复完成
   - 不发送无意义问题

## 数据安全

### 隐私保护

1. **本地存储**
   - 对话历史仅存储在您的浏览器
   - 不上传到服务器
   - 清除浏览器数据即可删除

2. **API 调用**
   - 仅发送必要信息（股票代码、问题）
   - 不收集个人信息
   - 符合隐私政策

3. **数据安全**
   - API 通信使用 HTTPS 加密
   - 不存储敏感金融数据
   - 不记录用户行为日志

## 故障排查

### 问题: AI 不响应

**检查清单**:
- [ ] 网络连接正常
- [ ] API 密钥配置正确
- [ ] 浏览器控制台无错误
- [ ] DeepSeek 服务可用

**解决方案**:
1. 刷新页面重试
2. 检查环境变量
3. 查看控制台错误
4. 联系技术支持

### 问题: 响应速度慢

**可能原因**:
- 网络延迟
- API 服务器负载高
- 问题过于复杂

**解决方案**:
1. 简化提问
2. 使用预设模板
3. 避免高峰期
4. 检查网络速度

### 问题: 对话历史丢失

**可能原因**:
- 清除了浏览器数据
- 切换了浏览器
- 使用了隐私模式

**解决方案**:
- 对话历史无法恢复
- 重新开始对话
- 避免清除浏览器数据
