## 常见问题

### Q1: 搜索很慢怎么办？

**排查步骤**:
1. 检查网络连接
2. 查看Network面板的API响应时间
3. 验证后端API是否正常
4. 检查是否有代理或VPN影响

### Q2: 搜索历史消失了？

**原因**:
- 浏览器清除了缓存
- 隐私模式下localStorage不持久化
- localStorage存储已满

**解决**:
- 避免清除浏览器数据
- 使用正常模式浏览
- 清理其他网站的localStorage

### Q3: 热门股票显示错误？

**排查**:
1. 检查 `/api/stocks/hot` API
2. 查看开发者控制台错误
3. 验证API返回格式

**注意**: 目前使用模拟数据，可能与真实行情不符

### Q4: 添加自选股失败？

**排查**:
1. 检查是否登录
2. 查看 `/api/watchlist` API
3. 验证数据库连接
4. 查看控制台错误日志
