## 📝 测试步骤

### 步骤1: 登录账号

1. 访问 https://www.alphapercept.com
2. 点击右上角"登录"按钮
3. 使用您的账号登录

### 步骤2: 进入自选股页面

1. 登录后，点击左侧菜单的"自选股"
2. 或直接访问：https://www.alphapercept.com/watchlist
3. 确认页面正常显示

### 步骤3: 添加股票（方式1 - 搜索选择）

1. 点击页面右上角的"添加股票"按钮
2. 在弹出对话框的搜索框中输入：`000001` 或 `平安银行`
3. 等待搜索结果出现
4. **点击搜索结果中的股票**（重要！）
5. 确认股票代码和名称已自动填入
6. （可选）填写买入成本、止损价、目标价
7. 点击"添加"按钮
8. 查看是否成功添加

### 步骤4: 添加股票（方式2 - 手动输入）

1. 点击"添加股票"按钮
2. **直接在股票代码框中输入**：`600519`
3. **直接在股票名称框中输入**：`贵州茅台`
4. （可选）填写其他信息
5. 点击"添加"按钮
6. 查看是否成功添加

### 步骤5: 查看自选股列表

1. 添加成功后，应该在列表中看到新添加的股票
2. 确认显示了股票代码、名称、价格等信息
3. 确认可以看到实时价格更新

### 步骤6: 编辑和删除

1. 找到列表中的任意股票
2. 点击"编辑"按钮，修改价格信息
3. 保存，确认修改成功
4. 点击"删除"按钮
5. 确认删除成功

## ❌ 常见问题

### 问题1: "添加"按钮是灰色的，无法点击

**原因**: 缺少必填项（股票代码或名称）

**解决方案**：
1. 检查股票代码框是否已填写
2. 检查股票名称框是否已填写
3. 可以通过搜索选择，或直接手动输入

### 问题2: 点击"添加"后没有反应

**可能原因**：
1. 网络问题 - 打开浏览器开发者工具查看Network标签
2. 未登录 - 确认右上角显示您的用户名
3. API错误 - 查看Console标签是否有错误

**解决方案**：
1. 刷新页面重试
2. 检查登录状态
3. 截图Console错误并反馈

### 问题3: 搜索不到股票

**原因**: 股票数据库可能不包含该股票

**解决方案**：
使用直接输入方式：
1. 股票代码：6位数字（如：000001）
2. 股票名称：完整名称（如：平安银行）

### 问题4: 显示"请先登录"

**原因**: Clerk认证session过期或未登录

**解决方案**：
1. 点击右上角"登录"按钮
2. 完成登录流程
3. 返回自选股页面

## 🔧 调试信息

### 浏览器开发者工具

按F12或右键→检查，打开开发者工具：

**Network标签**：
1. 添加股票时，应该看到：
   - POST /api/watchlist
   - Status: 200 OK（成功）
   - Status: 401（需要登录）
   - Status: 409（股票已存在）

2. 查看列表时，应该看到：
   - GET /api/watchlist
   - Status: 200 OK
   - Response包含watchlist数组

**Console标签**：
- 查看是否有错误信息
- 查看日志输出（以[WatchlistManager]开头）

### 请求示例

成功的POST请求：
```json
// Request
{
  "stockCode": "000001",
  "stockName": "平安银行",
  "buyPrice": 10.5,
  "stopLossPrice": 9.5,
  "targetPrice": 12.0,
  "notes": "测试"
}

// Response
{
  "success": true,
  "item": {
    "id": "xxx",
    "stockCode": "000001",
    "stockName": "平安银行",
    ...
  }
}
```
